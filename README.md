# A/B-тест для веб-приложения — дизайн, MDE, проверка на подглядывание и бизнес-рекомендация
Контекст: На смоделированных данных онлайн-кинотеатра («Синематика») спроектирован и проведён A/B-тест персонализированного рекомендательного блока на странице фильма. Оценено влияние на конверсию в просмотр (подписка/аренда). Рассчитан минимальный практически значимый эффект (MDE = 0.5 п.п.), определён необходимый размер выборки (~24 тыс. пользователей). Визуализирована проблема многократного подглядывания за p-value и обоснована опасность ранней остановки теста.
Стек: Python (pandas, numpy, statsmodels, matplotlib), z-тест пропорций, симуляция peeking.
Результат: Статистически значимый прирост конверсии с 4% до 4.5% (p=0.03, 95% CI [0.1, 0.9] п.п.). Дана рекомендация внедрить блок, что по прогнозу принесёт дополнительно ~600 тыс. руб. выручки в первый месяц.
